Python开源IP代理池proxy_pool

start

proxy_pool是一款国内使用率比较高的开源免费IP代理池项目。因为整体代码比较简单，所以打算fork下来，便于以后扩展使用。
首先我们来安装初始版本（v2.0.1）。
需要注意的是，因为依赖包的关系，proxy_pool目前只支持到python3.6，同时请尊许requirements中声明的版本，否则容易出问题。
另外安装前最好创建单独的虚拟环境，避免与其它项目起冲突。

ps:
requirements中少了redis-py，需要手动补上

SSDB

基础依赖安装好后，还需要安装外部数据库。proxy_pool默认使用SSDB或者Redis（社区有支持其它db的方案）。初级阶段，选取作者推荐的SSDB即可。

SSDB⽀持三种数据类型, 别分是 KV(key-value), Hashmap(map), Zset(sorted set)。
SSDB支持Redis网络协议, 所以你可以用Redis的客户端来连接SSDB服务器。Mac可以使用开源的Medis

ps:
如果遇到No identity found for signing错误不必惊慌。因为正式打包mac app是需要签名的（签名比较麻烦，需要购买并设置证书…），我们可以将pack.js中有关签名的部分去掉，生成一个未签名app来使用。
