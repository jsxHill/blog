更加轻松的使用node

版本管理

最好使用一款Node版本管理器，不仅可以进行多版本管理，卸载也更加方便，还可以避免一些全局安装带来的问题（比如Mac下npm i -g需要sudo权限）。easy life。

使用n

轻便实用，缺点是仅限Mac/Linux环境。

使用nvm（推荐）

跟n不同，nvm是独立于npm的外部shell脚本，使用限制更少，并且能够做到不同node版本使用不同的全局模块。
windows系统可以使用nvm-windows，效果是一样的。

gyp: No Xcode or CLT version detected

系统: macOS
原因：不明
解决方法：
重装
1. 输入xcode-select –print-path查看 command-line tools 的安装路径，不出意外显示的结果应该是/Library/Developer/CommandLineTools
2. 输入sudo rm -r -f /Library/Developer/CommandLineTools把 command-line tools 从系统移除掉
3. 最后输入xcode-select –install重新安装
